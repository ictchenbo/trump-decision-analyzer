### 修改数据库连接配置的方法
#### 1.  最推荐：通过`.env`文件修改（持久化配置）
编辑 `backend/.env` 文件，修改对应参数即可：
```ini
# MongoDB连接地址（支持带账号密码的格式）
MONGODB_URL=mongodb://username:password@your-database-host:27017
# 数据库名称
DB_NAME=your_custom_database_name
# 后端服务端口（可选）
PORT=8000
```

#### 2.  临时修改：通过环境变量临时覆盖配置
在启动后端服务前设置环境变量，Windows PowerShell示例：
```powershell
# 设置临时环境变量
$env:MONGODB_URL="mongodb://username:password@your-remote-host:27017"
$env:DB_NAME="new_database"

# 启动后端服务
python -m app.main
```

#### 3.  直接修改默认值（不推荐）
编辑 `backend/app/core/config.py` 文件，修改`Settings`类的默认参数，但这种方式会将硬编码配置提交到版本控制，仅适合本地快速调试。

---
