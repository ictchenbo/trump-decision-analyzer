#### 4.1.2 多因子权重评估模型
**功能**：量化五维因子的实时影响权重

| 因子 | 子指标 | 权重算法 | 更新频率 |
|------|--------|----------|----------|
| 地缘政治压力 | 海峡封锁强度 | (100-通行量%)/10 | 每日 |
| | 冲突烈度指数 | 军事事件频次×严重程度 | 实时 |
| 国内政治周期 | 支持率压力 | (40-当前支持率)×2.5 | 每周 |
| | 选举紧迫性 | 距离选举天数/30 | 每日 |
| 金融市场指标 | 市场恐慌指数 | VIX/10 | 实时 |
| | 股市压力 | 跌幅×2(负值取绝对值) | 实时 |
| 能源市场基本面 | 油价危机指数 | (当前油价-80)/4 | 实时 |
| | 供应缺口压力 | 缺口(百万桶)/100 | 每日 |
| 决策团队 | 约束机制指数 | 温和派官员占比×10 | 持续 |

**综合影响指数计算**：
```
综合指数 = Σ(因子得分 × 动态权重)

动态权重根据历史回归分析确定，示例：
- 选举前6个月：国内政治权重+30%
- 油价>100美元：能源市场权重+25%
- 股市暴跌>2%：金融市场权重+20%
```

#### 4.1.3 模式识别引擎
**功能**：识别四种决策触发模式

| 模式 | 识别规则 | 置信度计算 |
|------|----------|------------|
| **选举危机驱动型** | 支持率<40% AND 选举<180天 AND 外交政策突然鹰派化 | 历史相似度匹配 |
| **市场崩盘驱动型** | 股市跌幅>2% AND 国债收益率急升 AND 政策急转弯 | 时间序列异常检测 |
| **地缘压力过载型** | 油价>120 AND 盟友失控行为 AND 公开斥责盟友 | 事件关联分析 |
| **个人风格放大型** | 温和派官员离职 AND 不计后果冒险 AND 快速撤退 | 决策链分析 |
